# 雪球投资助手 (Xueqiu AI Studio)

基于 Chrome 扩展程序的现代化投资监控与资产管理工具。

## 核心特性

- **Liquid Glass UI**: 沉浸式磨砂玻璃设计，支持隐私模式。
- **多账户管理**: 支持配置多个投资账户，实时计算现金仓位与总市值。
- **股价波动预警**: 毫秒级判定股价波动，支持通过 NTFY 发送移动端推送。
- **决策辅助**: 集成价格网格计算器与仓位测算工具。

## 环境要求

- **Chrome 浏览器**: 支持 Manifest V3 的版本。
- **Node.js**: v16.0.0+ (仅用于运行单元测试)。
- **NTFY (可选)**: 用于接收手机端消息推送。

## 快速开始

### 1. 插件安装
1. 下载并解压本项目。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的 **"开发者模式"**。
4. 点击 **"加载已解压的扩展程序"**，选择本项目根目录。

### 2. 开发环境与迁移
如果你需要进行二次开发或在不同目录下迁移项目：

- **安装依赖**：在新目录下运行以下命令安装测试框架及开发工具（建议不要直接复制 `node_modules`）：
  ```powershell
  npm install
  ```
- **验证环境**：开发前建议运行单元测试以确保逻辑正常：
  ```powershell
  npm test
  ```
- **重新加载插件**：复制项目到新目录后，需在 `chrome://extensions/` 中删除旧条目，并重新加载新路径下的文件夹。
- **数据迁移**：由于 Extension ID 随路径变化，建议迁移前先执行“数据导出”，在新环境下“导入”以恢复配置。

### 3. 配置预警
- 在投资看板中勾选“预警”选项。
- 如需移动端推送，请在 `modules/config.js` 或设置面板中配置 NTFY 订阅主题。

## 注意事项

- **数据存储**: 所有数据均存储在浏览器的 `chrome.storage.local` 中，建议定期通过看板的“导出数据”功能进行备份。
- **安全验证**: 频繁刷新雪球数据可能触发雪球的安全验证（验证码），插件会自动发出安全警报。
